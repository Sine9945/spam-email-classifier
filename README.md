# Spam Email Classifier

A machine learning project that classifies emails as **Spam** or **Not Spam** using Natural Language Processing and classification models.

## Objective

Build an intelligent system that detects spam emails using text preprocessing, feature extraction, and machine learning techniques.

## Project Structure

```text
data/                -> Datasets used for training/testing
outputs/             -> Results, predictions, evaluation outputs
src/
   backend/          -> Flask API backend
      app.py
   frontend/         -> User interface
      index.html
   preprocessing/    -> Text cleaning and preprocessing
   features/         -> Feature extraction logic
   models/           -> Trained ML models / training scripts
README.md            -> Project documentation
```

## Features

* Email text classification
* Spam / Not Spam prediction
* Web interface for user input
* Flask backend API
* Machine learning based detection

## Tech Stack

* Python
* Flask
* Scikit-learn
* Pandas
* HTML / CSS / JavaScript
